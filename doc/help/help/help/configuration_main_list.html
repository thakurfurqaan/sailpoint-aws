System Setup and Global Configuration
You can configure system-wide default settings for the entire IdentityIQ product, including features such as identity mapping, account mapping, role configuration, scopes, certification performance, the Lifecycle Manager settings (if you have purchased that product), and more.
To set up up global configuration options for IdentityIQ, click the gear icon on the Navigation menu bar and select from the list of options:
• Global Configuration
• Compliance Manager
• Lifecycle Manager
For detailed information, refer to the IdentityIQ System Configuration Guide located in your IdentityIQ_InstallationDirectory\doc\pdf directory.
